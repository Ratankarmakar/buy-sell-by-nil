# Buy sell by Nil

A comprehensive candlestick pattern trading strategy for TradingView that detects 26+ patterns with advanced filters and risk management.

## 📊 Features

- **26+ Candlestick Patterns**: Engulfing, Morning/Evening Star, Hammer, Doji, Harami, Piercing, Dark Cloud, Three Soldiers/Crows, Kicker, Marubozu, Abandoned Baby, and more
- **Smart Filters**: Trend filter (SMA/EMA), volume confirmation, minimum body size
- **Risk Management**: ATR-based stop-loss, risk:reward take profit, emergency exit
- **Visual Tools**: Buy/sell arrows, trend background, SL/TP lines, performance table

## ⚙️ How to Use

1. Copy the code from `buy-sell-by-nil.pine`
2. Open TradingView Pine Editor
3. Paste the code and click "Add to Chart"
4. Configure patterns and settings as desired

## 📈 Settings Guide

- **Pattern Selection**: Enable/disable specific candlestick patterns
- **Filters**: Trend direction, volume confirmation, minimum body size
- **Risk Management**: ATR period, SL multiplier, risk:reward ratio
- **Confirmation**: Optional next-candle confirmation
